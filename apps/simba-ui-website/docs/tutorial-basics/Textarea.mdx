The Textarea component allows you to easily create multi-line text inputs.

## Import

```js
import { MyTextarea } from "tailwind-simba-ui";
```

## Usage

export const BasicUsage = () => (
  <div className="border-2 rounded-xl mb-4">
    <div className="m-2">

    </div>

## Resize behavior

export const TextareaResize = ({ resize }) => (
  <div className="border-2 rounded-xl mb-4">
    <div className="m-2">
      <Tabs>
        <TabItem value="vertical" label="Vertical">
          <MyTextarea resize="vertical" placeholder="Vertical" />
        </TabItem>
        <TabItem value="horizontal" label="Horizontal">
          <MyTextarea resize="horizontal" placeholder="Horizontal" />
        </TabItem>
        <TabItem value="both" label="Both">
          <MyTextarea resize="both" placeholder="Both" />
        </TabItem>
        <TabItem value="none" label="None">
          <MyTextarea resize="none" placeholder="None" />
        </TabItem>
      </Tabs>
    </div>
  </div>
);

You can add resize prop to change the Textarea resize behavior. **Horizontal**, **both** or **none**.
