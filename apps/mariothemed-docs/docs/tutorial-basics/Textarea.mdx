The Textarea component allows you to easily create multi-line text inputs.

## **Import**

```js
import { Textarea } from "@bienmag/mariothemed-ui";
```

## **Controlled Textarea**

export const ControlledTextarea = function Controlled({ onChange }) {
  const [value, setValue] = useState("");
  return (
    <div className="border-2 rounded-xl mb-4">
      <div className="m-2">
        <p className="m-2"> Value : {value}</p>
        <Textarea
          onChange={(e) => setValue(e.target.value)}
          placeholder="Here is a sample placeholder"
        />
      </div>
    </div>
  );
};

<ControlledTextarea></ControlledTextarea>

```js
function Example() {
  const [value, setValue] = React.useState("");
  const handleChange = (event) => setValue(event.target.value);

  return (
    <div>
      <p>Value: {value}</p>
      <Textarea
        onChange={handleChange}
        placeholder="Here is a sample placeholder"
      />
    </div>
  );
}
```

## **Resize behavior**

You can add resize prop to change the Textarea resize behavior. **Horizontal**, **both**, **vertical** or **none**.

export const TextareaResize = ({ resize }) => (
  <div className="border-2 rounded-xl mb-4">
    <div className="m-2">
      <Tabs>
        <TabItem value="horizontal" label="Horizontal">
          <Textarea resize="horizontal" placeholder="Horizontal" />
        </TabItem>
        <TabItem value="both" label="Both">
          <Textarea resize="both" placeholder="Both" />
        </TabItem>
        <TabItem value="vertical" label="Vertical">
          <Textarea resize="vertical" placeholder="Vertical" />
        </TabItem>
        <TabItem value="none" label="None">
          <Textarea resize="none" placeholder="None" />
        </TabItem>
      </Tabs>
    </div>
  </div>
);

<TextareaResize></TextareaResize>

```js
<div>
  <Textarea resize="horizontal" />
  <Textarea resize="both" />
  <Textarea resize="vertical" />
  <Textarea resize="none" />
</div>
```

## **Changing the color of the textarea**

Use the color prop to change the visual style of the Textarea. You can set the value to **yellow**, **red**, **blue**, or **green**.

export const TextareaColors = ({ color }) => (
  <div className="flex flex-col border-2 rounded-xl mb-4">
    <div className="m-2">
      <Textarea color="yellow" placeholder="Yellow textarea" />
    </div>
    <div className="m-2">
      <Textarea color="red" placeholder="Red textarea" />
    </div>
    <div className="m-2">
      <Textarea color="blue" placeholder="Blue textarea" />
    </div>
    <div className="m-2">
      <Textarea color="green" placeholder="Green textarea" />
    </div>
  </div>
);

<TextareaColors></TextareaColors>

```js
<div>
  <Textarea variant="outline" placeholder="Outline" />
  <Textarea variant="filled" placeholder="Filled" />
  <Textarea variant="flushed" placeholder="Flushed" />
  <Textarea variant="unstyled" placeholder="Unstyled" />
</div>
```
