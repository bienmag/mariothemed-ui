Input component is a component that is used to get user input in a text field.

### **Import**

```js
import { Input } from "@bienmag/mariothemed-ui";
```

### **Usage**

Here's a basic usage example of the Input component:

export const BasicUsage = ({ size }) => (
  <div className="border-2 rounded-xl mb-4">
    <div className="m-2">
      <Input placeholder="Basic usage" />
    </div>
  </div>
);

<BasicUsage></BasicUsage>

```js
<Input placeholder="Basic usage" />
```

### **Input Sizes**

Use the size prop to change the size of the input. You can set the value to **xs**, **sm**, **md**, or **lg**.

export const InputSizes = ({ size }) => (
  <div className="flex flex-col border-2 rounded-xl mb-4">
    <div className="m-2">
      <Input size="xs" placeholder="extra small size"></Input>
    </div>
    <div className="m-2">
      <Input size="sm" placeholder="small size"></Input>
    </div>
    <div className="m-2">
      <Input size="md" placeholder="medium size"></Input>
    </div>
    <div className="m-2">
      <Input size="lg" placeholder="large size"></Input>
    </div>
  </div>

## **Changing the appearance of the input**

The input component comes in 4 variants: **outline**, **filled**, **flushed**, or **unstyled**. Pass the variant prop and set it to one of these values.

export const InputVariants = ({ variant }) => (
  <div className="flex flex-col border-2 rounded-xl mb-4">
    <div className="m-2">

## **Controlled input**

export const ControlledInput = function Controlled({ onChange }) {
  const [value, setValue] = useState("");
  return (
    <div className="flex flex-col border-2 rounded-xl mb-4">
      <div className="m-2">
        <p className="m-2"> Value : {value}</p>
        <Input
          color="yellow"
          variant="filled"
          value={value}
          onChange={(e) => setValue(e.target.value)}
        ></Input>
      </div>
    </div>
  );
};
