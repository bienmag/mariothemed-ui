# Input

Input component is a component that is used to get user input in a text field.

### Import

```js
import { MyInput } from "tailwind-simba-ui";
```

### Usage

export const BasicUsage = ({ size }) => (
  <div className="border-2 rounded-xl mb-4">
    <div className="m-2">
      <MyInput placeholder="Basic usage" />
    </div>
  </div>
);

Here's a basic usage example of the Input component:

<BasicUsage></BasicUsage>

```js
<MyInput placeholder="Basic usage" />
```

### Input Sizes

export const InputSizes = ({ size }) => (
  <div className="flex flex-col border-2 rounded-xl mb-4">
    <div className="m-2">
      <MyInput size="xs" placeholder="extra small size"></MyInput>
    </div>
    <div className="m-2">
      <MyInput size="sm" placeholder="small size"></MyInput>
    </div>
    <div className="m-2">
      <MyInput size="md" placeholder="medium size"></MyInput>
    </div>
    <div className="m-2">
      <MyInput size="lg" placeholder="large size"></MyInput>
    </div>
  </div>
);

Use the size prop to change the size of the input. You can set the value to **xs**, **sm**, **md**, or **lg**.

<InputSizes></InputSizes>

```js
<div>
  <MyInput size="xs" />
  <MyInput size="sm" />
  <MyInput size="md" />
  <MyInput size="lg" />
</div>
```

## Changing the appearance of the input

export const InputVariants = ({ variant }) => (
  <div className="flex flex-col border-2 rounded-xl mb-4">
    <div className="m-2">
      <MyInput variant="outline" placeholder="Outline" />
    </div>
    <div className="m-2">
      <MyInput variant="filled" placeholder="Filled" />
    </div>
    <div className="m-2">
      <MyInput variant="flushed" placeholder="Flushed" />
    </div>
    <div className="m-2">
      <MyInput variant="unstyled" placeholder="Unstyled" />
    </div>
  </div>
);

The input component comes in 4 variants: **outline**, **filled**, **flushed**, or **unstyled**. Pass the variant prop and set it to one of these values.

<InputVariants></InputVariants>

```js
<div>
  <MyInput variant="outline" placeholder="Outline" />
  <MyInput variant="filled" placeholder="Filled" />
  <MyInput variant="flushed" placeholder="Flushed" />
  <MyInput variant="unstyled" placeholder="Unstyled" />
</div>
```
