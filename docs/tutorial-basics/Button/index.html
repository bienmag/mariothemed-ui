Button

Button component is used to trigger an action or event, such as submitting a form, opening a Dialog, canceling an action, or performing a delete operation.

Import​

import { Button } from "tailwind-simba-ui";

Usage​

<Button color="yellow">Button</Button>

Button Sizes​

Use the size prop to change the size of the button. You can set the value to xs, sm, md, or lg.

<div>
  <Button color="yellow" size="xs">
    Button
  </Button>
  <Button color="yellow" size="sm">
    Button
  </Button>
  <Button color="yellow" size="md">
    Button
  </Button>
  <Button color="yellow" size="lg">
    Button
  </Button>
</div>

Button variants​

Use the variant prop to change the visual style of the Button. You can set the value to solid, ghost, outline, or link.

<div>
  <Button color="yellow" variant="solid">
    Button
  </Button>
  <Button color="yellow" variant="outline">
    Button
  </Button>
  <Button color="yellow" variant="ghost">
    Button
  </Button>
  <Button color="yellow" variant="link">
    Button
  </Button>
</div>

Button color​

Use the color prop to change the visual style of the Button. You can set the value to yellow, red, blue, or green.

<div>
  <Button color="yellow">Button</Button>
  <Button color="red">Button</Button>
  <Button color="blue">Button</Button>
  <Button color="green">Button</Button>
</div>

Button with icon​

You can add an icon to your button. Use the leftIcon prop to place the image on the left of the button and rightIcon to place the image on the right of the button.

info

Please note that the icon should be a react element and not a string!

<div>
  <Button color="yellow" leftIcon={<Icon />}>
    Button
  </Button>
  <Button color="red" rightIcon={<Icon />}>
    Button
  </Button>
</div>

Customize your button​

danger

CHECK THIS!

You can customize your button with className OR override the styles provided with the ui library

<Button size="xs" height="120px"/> // will override the height set in "xs"
<Button className="w-40 h-20 flex justify-center items-center" size="xs" color="yellow"/>
